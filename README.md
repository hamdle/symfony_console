# symfony_console
Test symfony console code from DrupalCon Vienna: Tour of the 35 Symfony Components

https://www.youtube.com/watch?v=TxKkYuNmoK8

#### How to run
```
composer install
php command.php
```

#### Available commands

* greet

#### Command example

```
php command.php greet Eric --yell
```

Output: HEY ERIC
